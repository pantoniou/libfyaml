libfyaml API
============

This section groups the public libfyaml API into smaller subsystem pages that
work better both in HTML and as section-3 manual pages.

Subsystem Pages
---------------

* :doc:`libfyaml-core`
* :doc:`libfyaml-misc`

See also
--------

The generated manual pages are:

* :manpage:`libfyaml-core(3)` for parsing, emitting, path execution, document
  traversal, document building, and composition.
* :manpage:`libfyaml-misc(3)` for utilities, allocators, threading, hashing,
  alignment, endianness, variable-length size encoding, and atomics.
